## Data Setup

The dataset is too large for GitHub. Download it manually:

1. Visit [Dubai Pulse - DLD Transactions](https://www.dubaipulse.gov.ae/data/dld-transactions/dld_transactions-open)
2. Download `transactions.csv`
3. Place it in the `data/` folder

Alternatively, run:
```bash
# If direct download link works
wget -O data/transactions.csv "https://www.dubaipulse.gov.ae/dataset/.../download/transactions.csv"
```
